# Visual Studio Code Keyboard Shortcuts (Windows)

| Action                                      | Shortcut                    |
|----------------------------------------------|-----------------------------|
| Show Explorer                               | Ctrl + Shift + E            |
| Show Search                                 | Ctrl + Shift + F            |
| Show Source Control                         | Ctrl + Shift + G            |
| Show Run & Debug                            | Ctrl + Shift + D            |
| Show Extensions                             | Ctrl + Shift + X            |
| Toggle Sidebar Visibility                   | Ctrl + B                    |
| Focus into Side Bar                         | Ctrl + 0                    |
| Move Focus between Editor Groups            | Ctrl + 1 / Ctrl + 2 / ...   |
| Split Editor (horizontal)                   | Ctrl + \                    |
| Focus into Editor Group (Left/Right)        | Ctrl + K then Left/Right    |
| Close Editor                                | Ctrl + W                    |
| Reopen Closed Editor                        | Ctrl + Shift + T            |
| Move Editor into Next Group                 | Ctrl + Shift + Page Down    |
| Move Editor into Previous Group             | Ctrl + Shift + Page Up      |
| Maximize/Minimize Editor                    | Ctrl + Shift + Enter        |
| Duplicate Line/Selection (Down)             | Shift + Alt + Down Arrow         |
| Duplicate Line/Selection (Up)               | Shift + Alt + Up Arrow           |
