# Visual Studio Code Keyboard Shortcuts (Mac)

| Action                                      | Shortcut                          |
|----------------------------------------------|-----------------------------------|
| Show Explorer                               | Cmd + Shift + E                   |
| Show Search                                 | Cmd + Shift + F                   |
| Show Source Control                         | Cmd + Shift + G                   |
| Show Run & Debug                            | Cmd + Shift + D                   |
| Show Extensions                             | Cmd + Shift + X                   |
| Toggle Sidebar Visibility                   | Cmd + B                           |
| Focus into Side Bar                         | Cmd + 0                           |
| Move Focus between Editor Groups            | Cmd + 1 / Cmd + 2 / ...           |
| Split Editor (horizontal)                   | Cmd + \                           |
| Focus into Editor Group (Left/Right)        | Cmd + K then Left/Right Arrow     |
| Close Editor                                | Cmd + W                           |
| Reopen Closed Editor                        | Cmd + Shift + T                   |
| Move Editor into Next Group                 | Ctrl + Shift + Page Down          |
| Move Editor into Previous Group             | Ctrl + Shift + Page Up            |
| Maximize/Minimize Editor                    | Cmd + Shift + Enter               |

Duplicate Line/Selection (Down): Shift + Option + Down Arrow  
Duplicate Line/Selection (Up): Shift + Option + Up Arrow
